    GetParentDefinition(EventInfo)

    Gets the most derived override of this event up the inheritance chain.

    Declaration
    public static EventInfo GetParentDefinition(this EventInfo eventInfo)
    Parameters
    Type Name Description
    EventInfo eventInfo

    The event to check.
    Returns
    Type Description
    EventInfo

    The most derived definition of the specified eventInfo in a base class.
    Remarks

    Returns override of the specified event in the most derived class that is a base class of the declaring type of the specified eventInfo if it exists. If no override exists prior to this one in the inheritance chain, this method will return the same as GetBaseDefinition(EventInfo).
