    Syntax
    protected class ProjectedPropertyInfo<TContext> : DelegatingPropertyInfo, ICustomAttributeProvider, _MemberInfo, _PropertyInfo, CustomReflectionContextBase.IProjector<TContext>, CustomReflectionContextBase.IProjector where TContext : CustomReflectionContextBase
    Type Parameters
    Name Description
    TContext

    The type of the reflection context. Must inherit from CustomReflectionContextBase.
    Remarks

    This projector is intended to be used as a base class for projectors in derived reflection contexts. It adds no functionality to the underlying object, other than ensuring that all types and members returned from the object are also mapped in the same reflection context.
