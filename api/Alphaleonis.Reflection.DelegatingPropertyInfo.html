    Remarks

    Derive from this type and override only those members you have to provide customization in.

    DelegatingPropertyInfo derives from PropertyInfo and implements most of the properties and methods of PropertyInfo. For each member it implements, DelegatingPropertyInfo automatically delegates to the corresponding member of the internal PropertyInfo object which was supplied as the argument to the constructor. This internal PropertyInfo is exposed to deriving classes by the UnderlyingProperty property.
