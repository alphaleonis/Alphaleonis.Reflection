    Syntax
    public class DelegatingAssembly : Assembly, _Assembly, IEvidenceFactory, ICustomAttributeProvider, ISerializable
    Remarks

    Derive from this type and override only those members you have to provide customization in.

    DelegatingAssembly derives from Assembly and implements most of the properties and methods of Assembly. For each member it implements, DelegatingAssembly automatically delegates to the corresponding member of the internal Assembly object which was supplied as the argument to the constructor. This internal Assembly is exposed to deriving classes by the UnderlyingAssembly property.
