    Remarks

    Derive from this type and override only those members you have to provide customization in.

    DelegatingConstructorInfo derives from ConstructorInfo and implements most of the properties and methods of ConstructorInfo. For each member it implements, DelegatingConstructorInfo automatically delegates to the corresponding member of the internal ConstructorInfo object which was supplied as the argument to the constructor. This internal ConstructorInfo is exposed to deriving classes by the UnderlyingConstructor property.
