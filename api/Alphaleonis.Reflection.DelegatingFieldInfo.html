    public class DelegatingFieldInfo : FieldInfo, ICustomAttributeProvider, _MemberInfo, _FieldInfo
    Remarks

    Derive from this type and override only those members you have to provide customization in.

    DelegatingFieldInfo derives from FieldInfo and implements most of the properties and methods of FieldInfo. For each member it implements, DelegatingFieldInfo automatically delegates to the corresponding member of the internal FieldInfo object which was supplied as the argument to the constructor. This internal FieldInfo is exposed to deriving classes by the UnderlyingField property.
